# 🚀 CSE Project Archive System

> A scalable web-based application designed to manage, organize, and retrieve Computer Science student projects through a centralized digital repository.

---

## 📌 Overview

The **CSE Project Archive System** is a web application that helps students and faculty efficiently store, manage, and access academic projects.
It improves accessibility, avoids duplication, and provides a structured repository for academic use.

---

## 🎯 Problem Statement

Students often face challenges such as:

* Difficulty accessing previous projects
* Duplicate project submissions
* Lack of a centralized system

👉 This system solves these issues by providing a **centralized and searchable platform**.

---

## 💡 Key Features

* 🔐 User Authentication (Register & Login)
* 📂 Project Submission & Management
* 🔍 Search Projects (Name / Roll No / Batch)
* 📊 User Dashboard
* 🗃 Centralized Repository
* ⚡ Simple and User-Friendly Interface

## ⚙️ Installation & Setup

```bash
git clone https://github.com/sandipkumarnke9-design/CSE_Project_Archive_System.git
cd CSE_Project_Archive_System
```

### Run Locally:

1. Install XAMPP
2. Copy project folder into `htdocs`
3. Create database named `cse_projects`
4. Import `database.sql` (if available)
5. Update database configuration in `database/db.php`
6. Start Apache & MySQL
7. Open in browser:
   http://localhost/CSE_Project_Archive_System/public/
